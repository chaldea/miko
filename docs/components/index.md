# Component Reference <Badge type="warning" text="preview" />

::: info Reserved
The per-component reference is **not published yet**. Miko's component libraries
(`Miko.Bootstrap` and `Miko.Ionic`) are still evolving, and their parameters / APIs are
not yet stable. To avoid documenting interfaces that are about to change, this section is
intentionally left as a placeholder and will be filled in once the component APIs settle.
:::

## What will live here

When the component APIs stabilize, this section will document the ready-made Razor
component libraries, including:

### Miko.Bootstrap

Bootstrap-inspired components such as: Accordion, Alert, Badge, Breadcrumb, Button, Card,
Carousel, Dropdown, Form controls, Grid, Icon, List, Modal, Navbar, Pagination, Popover,
Progress, Spinner, Tab, Table, Toast, and Tooltip — each with its parameters, slots, and
examples.

### Miko.Ionic

Ionic-style components and layouts (tabs, side menu, Ionicons) used by the `tabs` and
`sidemenu` app templates.

## In the meantime

- Browse the [Bootstrap example gallery](/examples#bootstrap), which renders most of the
  Bootstrap components and is the best current reference for how they look and compose.
- Read the source under
  [`src/Miko.Bootstrap/Components`](https://github.com/chaldea/miko/tree/main/src/Miko.Bootstrap/Components)
  and
  [`src/Miko.Ionic/Components`](https://github.com/chaldea/miko/tree/main/src/Miko.Ionic/Components).
- For building your own components, see [Razor Components](/guide/razor-components).
